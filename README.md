# Daily-List
Daily List Kebutuhan Sehari-hari App adalah aplikasi sederhana untuk membantu pengguna mencatat dan mengelola daftar kebutuhan harian secara terorganisir. Aplikasi ini memudahkan pengguna dalam menyiapkan, memantau, dan memastikan kebutuhan sehari-hari terpenuhi dengan lebih efisien dan praktis.
